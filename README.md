# Polynomial Calculator
```
A simple, browser‑based **Polynomial Calculator** built using **HTML, CSS, and JavaScript**.  
It allows users to input polynomial coefficients and evaluate the polynomial at a given value of x.  
The interface is clean, responsive, and easy to use.
VISIT THE APPLICATION : 
**https://adithi2006.github.io/polynomial-calculator/**
---

## ✨ Features
- **Polynomial evaluation** using Horner’s method for precise results.  
- **Professional UI** with modern styling and hover effects.  
- **Error handling** for invalid or missing inputs.  
- **Single‑file setup** — no external libraries required.  
- Works seamlessly with **VS Code + Live Server**.

## 🚀 How to Run
1. Save the file as `polynomial_calculator.html`.  
2. Open it in your browser:
   - Right‑click → **Open with Live Server** (VS Code extension), or  
   - Double‑click the file to open directly in Chrome/Edge.  
3. Enter coefficients (comma‑separated) and a value for x.  
4. Click **Calculate** → the result appears instantly.
